# How To Design An iOS App Icon That Gets Tapped

The icon has two jobs: get noticed, then feel trustworthy.

## Start With Recognition

- design for small size first
- use one strong mark
- remove detail until the icon still reads at a glance

## Judge It In Context

- compare it against real App Store competitor grids
- check it against both light and dark surroundings
- make sure the symbol still feels clear after masking

## Make It Match The Product

- the icon color story should connect to the app itself
- screenshots, onboarding, and the icon should feel like the same product
- if the app is calm and useful, the icon should not feel like a mobile game

## Common Mistakes

- too many elements
- weak contrast
- relying on text
- designing for a Dribbble shot instead of App Store search
- exporting art that reveals ugly edges once masked
