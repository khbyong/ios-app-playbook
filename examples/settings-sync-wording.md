# Settings Sync Wording

## Problem

A settings screen used a button labeled `Sync Now`, but the underlying action only refreshed status. Users could read it as upload, restore, or force merge.

## Weak Version

- title: `iCloud Sync`
- button: `Sync Now`
- supporting text: `Keeps your data synced.`

## Stronger Version

- title: `iCloud Backup & Sync`
- button: `Refresh Status`
- supporting text: `Daytab syncs automatically when iCloud is available. Refresh Status checks the current iCloud state. It does not erase local data or manually restore a backup.`

## Why It Is Better

- the button matches what the action actually does
- the copy reduces fear around destructive outcomes
- the wording makes room for a separate restore path later
